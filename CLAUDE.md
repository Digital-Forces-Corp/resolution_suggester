# Project Instructions

## Before committing

Update the `$Version` variable in `resolutions_suggester.ps1` to the current local ISO 8601 date-time with UTC offset (e.g. `2026-04-09T13:20-05:00`).

## After pushing

Remind the user that GitHub's raw CDN (`raw.githubusercontent.com`) can lag up to 5 minutes before serving the new version.
